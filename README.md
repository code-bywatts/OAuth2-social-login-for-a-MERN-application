# OAuth2 Social Login for a MERN Application by VASU WATTS
Implemented OAuth2 social login for a MERN application using Node.js (Express) and MongoDB, enabling users to authenticate with Google and Facebook. Support the Authorization Code flow with PKCE, map external identities to local user profiles, and establish a secure session (JWT in httpOnly cookies). 

# Backend:
Node.js & Express.js
MongoDB & Mongoose
JWT (jsonwebtoken)
bcryptjs
OAuth2 with PKCE
Axios

# Frontend:

React 18
Vite
React Router v6
Axios
Context API
